# Number Guessing Game (Java)

A simple and interactive number guessing game built in Java.  
The program generates a random number between 1 and 100, and the player must guess it using the hints provided (too high / too low). The game includes limited attempts, scoring, and the option to play multiple rounds.

## Features
- Random number generation (1–100)
- Hints: too high / too low / correct
- Limited number of attempts
- Score calculation
- Option to replay
- Beginner-friendly Java code

## How to Run
```bash
javac NumberGame.java
java NumberGame
